# SA Skills

AWS Solutions Architect skills for Claude Code.

## aws-sa-toolkit

SA daily workflow engine covering:

- **Tech Sharing** — presentation production (research → draft → PPT/HTML)
- **PoC Design** — customer proposal design with architecture and cost estimation
- **Meeting Minutes** — voice/text to structured Markdown minutes
- **Stakeholder Communication** — internal emails, resource requests, weekly updates
- **Activity Logging** — standardized SA activity records (GenAI / Regular Key / Migration)

### Installation

```bash
cp -r aws-sa-toolkit/ ~/.claude/skills/aws-sa-toolkit/
```

### Note

`SA-PPT-Template.pptx` is not included in this repo. Place your own SA PPT template at:
```
~/.claude/skills/aws-sa-toolkit/references/SA-PPT-Template.pptx
```
